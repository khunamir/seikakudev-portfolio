These are some of the details of this project

  • Tech Stack :
    • HTML
    • CSS
    • Javascript
    • C# / .Net 5.0
  • This web application is build with the purpose of aiding users to keep track of worukouts with a web journaling system :
    • CRUD operation for workouts
    • Searching and pagination on displayed workout
    • User management with IdentityUser (in development)
    • Authentication with JWT and session for validating JWT on subsequent request
    • Integration with local MSSQL database
